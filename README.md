# H1 Projeto piloto no node.js 

Este projeto foi desenvolvido no Visual Studio Code, recomendo usar este ambiente também. Basta clonar a pasta, ter o node.js instalado, MySQL no ar que o aplicativo deve rodar normalmente.

Para executar, vá ao terminal e execute
```
node app.js
```
Usa banco de dados MySQL. Para criar o banco de dados, use o script que está na raiz (script.sql)

Possui login, cuja senha é gerada no programa GeraSenha.js

O sistema consiste em um CRUD, com cadastro, listagem, exclusão, busca e edição na tabela de funcionários.

O renderizador de páginas usado foi o EJS.
